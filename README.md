# Personal Website Template

A modern, responsive personal website template built with HTML, CSS, and JavaScript. This template features a clean design, smooth animations, and is fully customizable to showcase your personal information, skills, and projects.

## Features

- Responsive design that works on all devices
- Modern and clean UI with smooth animations
- Mobile-friendly navigation with hamburger menu
- Sections for:
  - Hero/Introduction
  - About Me
  - Skills with progress bars
  - Projects showcase
  - Contact form
- Smooth scrolling navigation
- Intersection Observer for scroll animations
- Contact form with validation
- Social media links
- Skills progress animation

## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Customize the content to match your personal information

## Customization

### Personal Information
1. Open `index.html` and replace:
   - "Your Name" with your actual name
   - Update the subtitle in the hero section
   - Modify the about section text
   - Update education, experience, and location information

### Projects
1. In the projects section, replace the placeholder images with your project screenshots
2. Update project titles and descriptions
3. Add your project links (GitHub and live demo)

### Skills
1. Modify the skills section to match your expertise
2. Adjust the progress bar percentages to reflect your skill levels
3. Add or remove skill cards as needed

### Contact Information
1. Update the contact section with your:
   - Email address
   - Phone number
   - Location
   - Social media links

### Styling
1. Open `styles.css` to customize:
   - Colors (search for color codes like #3498db)
   - Font sizes
   - Spacing
   - Animations
